# Portfolio-pessoal
Este é o meu Portfolio Pessoal.
Este portfólio foi desenvolvido para apresentar minha evolução, minhas habilidades e meus projetos na área de desenvolvimento e tecnologia. A estrutura do site foi construída com foco em clareza, modernidade e boa experiência de navegação, utilizando um layout totalmente responsivo para garantir compatibilidade com todos os dispositivos.

O design segue uma estética moderna com elementos neon, efeitos de profundidade e cartões com aparência de vidro, destacando um estilo visual voltado para tecnologia. Cada seção foi planejada para demonstrar minha organização, meu conhecimento e minha capacidade de criar interfaces funcionais e bem estruturadas.

O conteúdo do portfólio inclui informações sobre minha trajetória, minhas habilidades técnicas, meus projetos e minhas experiências em aprendizagem, oferecendo uma visão completa do meu desenvolvimento na área. O objetivo é apresentar de forma direta e profissional o que estou construindo e como estou evoluindo como desenvolvedor.
